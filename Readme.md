# DevConf 2026

This is my first assignment for Programming Hero.

I built this project using only HTML and CSS by following the provided Figma design. The website is about a fictional developer conference called *DevConf 2026*.

## Sections

- Navbar
- Hero Banner
- Meet the Speakers
- Pricing
- Hackathon (AI Generated Section)
- Footer

## AI Challenge

For the AI challenge, I created the **Hackathon** section using AI, following the assignment instructions. The prompt I used is available in the `PROMPTS.md` file.

## Technologies Used

- HTML5
- CSS3

## Project Structure

The project is organized in a simple way to make the code easy to understand.

- `index.html` contains the complete structure of the website.
- `style.css` contains all the styling and layout of the website.
- The `assets` folder contains all the images and other required resources used in the project.
- The `UI` folder contains the original Figma design (*Before*) and my final implementation (*After*).
- I added comments in both the HTML and CSS files so that each section can be found and understood easily.
- `ExplainingConcepts.md` contains my written explanation of the required HTML and CSS concepts as an alternative to the optional explanation video.
